BASE macht Pause

Zurzeit werden keine Tarife unter der Marke BASE angeboten.

Was wird aus BASE?

Telefónica Deutschland bleibt seiner Mehrmarkenstrategie treu. Wie bereits in der Vergangenheit wird die Marke BASE das Portfolio von Telefónica Deutschland erweitern, wenn dies aufgrund einer aktuellen Marktsituation sinnvoll erscheint.

Derzeit ist das nicht der Fall, weshalb wir BASE vorläufig nicht aktiv vermarkten. Sollte eine Vermarktung strategisch wieder zielführend sein, wird das Unternehmen die Marke BASE entsprechend einsetzen.
